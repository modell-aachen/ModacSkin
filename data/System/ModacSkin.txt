---++ Usage

---+++ ModacSideBar

If there is a topic "ModacSideBar" within a web, it will be used to define the web's sidebar, otherwise the default sidebar defined in "System.ModacSideBarExample" 
will be used. So if you want to create a web-specific sidebar just create a topic "ModacSideBar" within this web.

---+++ ModacBottomBar

Same procedure as described in chapter "ModacSideBar" above: If there's a topic "ModacBottomBar", it defines the web-specific bottom bar otherwise the topic
"System.ModacBottomBar" is taken.

---+++ ModacTopBar

Same story as in "ModacSideBar" and "ModacBottomBar": Either there is a topic "ModacTopBar" within the web or "System.ModacTopBar" is used.

---++ Installation Instructions

To enable ModacSkin you have set the following variables either in WebPreferences (to enable ModacSkin within a single web) or
in Main/SitePreferences to enable the skin for the whole wiki.  

<pre>
   &#42; Set SKIN = modac

   &#42; Set FOSWIKI_LAYOUT_URL = &#37;PUBURLPATH&#37;/&#37;SYSTEMWEB&#37;/ModacSkinTheme/layout.css
   &#42; Set FOSWIKI_STYLE_URL = &#37;PUBURLPATH&#37;/&#37;SYSTEMWEB&#37;/ModacSkinTheme/style.css
   &#42; Set FOSWIKI_COLORS_URL = &#37;PUBURLPATH&#37;/&#37;SYSTEMWEB&#37;/ModacSkinTheme/colors.css

   &#42; Set FOSWIKI_PROJECT_URL =
</pre>

If you want to make custom modifications you can set FOSWIKI_PROJECT_URL to your css, otherwise set it to nothing.
